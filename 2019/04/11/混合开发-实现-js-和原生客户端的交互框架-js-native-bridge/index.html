基础用法

iOS，Android 客户端的混合开发，避免不了 js 和 native 之间的交互，一些常用的 js-bridge 库实现都是只支持一种系统。
js 调用 native 端的接口要简单，且一个函数就能调用 iOS 和 Android 两个系统，并且尽量模块化，在存在大量 native 接口的情况下便于维护这些函数。
